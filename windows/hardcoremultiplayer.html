My first attempt at Minecraft Java Programming: A hardcore challenge plugin designed for co-op Minecraft servers. All players are tied to a single life: if one dies, the entire server resets. This plugin introduces high tension and a roguelike progression loop, complete with automatic resets, world generation logic, and persistent attempt tracking. Built using the Spigot API with lots of tutorials and trial and error.

 

Key Features:

  • Shared Death System: If one player dies, everyone dies. Simple. Brutal. Memorable.
  • World Auto-Reset: Server fully resets and regenerates the world with fresh seed and spawn logic.
  • Attempt Tracking: Persistent run counter displayed in-game using scoreboards.
  • Session Debug Tools: Commands like /hc reset, /hc info, and /hc enable for quick testing and control.
  • Custom Spawn Behaviour: Players are safely teleported in after reset with optional delay and effects.
  • Discord Webhook Integration: Sends run summaries and death messages to a configured Discord server.
  • Server-Friendly: Works across local or hosted setups, tested with Paper and Spigot.
  • Lightweight and Reliable: No external dependencies, minimal config, fast world handling.
