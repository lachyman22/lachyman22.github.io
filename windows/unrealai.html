AI Navigation System [2024]

Student Project – Advanced Games Programming

 

My Contributions:

  • Implemented custom Navigation and Cover Node actor class in C++ to create a connected node graph for AI traversal.
  • Built logic for enemies to path between connected nodes, choose random patrol paths, or chase the player using node-based line-of-sight.
  • Wrote C++ systems for finding the nearest viable cover node based on distance and visibility when the AI is in an “injured” or “evading” state.
  • Created a multiplayer-safe C++ state machine with network-replicated transitions for modes like Patrol, Evade, Hiding, and Dead.
  • Integrated AI state and health info with the enemy HUD to reflect live combat status, ammo counts, and regeneration state.
